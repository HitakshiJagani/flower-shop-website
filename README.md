# 🌸 Flower Shop Website

A responsive and modern flower shop website built using HTML, CSS, and JavaScript.  
It showcases products, customer reviews, and a contact form for a complete user experience.

## 🚀 Features
- Responsive navigation bar
- Home, About, Products, Reviews, and Contact sections
- Product cards with pricing and discounts
- Customer reviews section
- Contact form UI
- Clean and modern design

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript

## ▶️ How to Run
1. Clone or download the repository
2. Open `index.html` in your browser
